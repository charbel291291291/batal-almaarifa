# 🏆 بطل المعرفة

مسابقة معلومات عربية سريعة الإيقاع — جولات متعددة، جرس، سلاسل ذهبية، وبطل واحد.
لعبة ويب أصلية بالكامل (اسم، هوية، آليات، نصوص، أصوات) بواجهة RTL عربية.
تتوفر الواجهة وبنك الأسئلة بالكامل بثلاث لغات: العربية والإنجليزية والفرنسية، مع تبديل RTL/LTR تلقائياً.

## التشغيل

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # بناء الإنتاج
```

## ما الموجود الآن

- **حفلة محلية (2–6 لاعبين)** على شاشة واحدة، جرس بمفاتيح الأرقام 1–6 أو باللمس.
- **5 جولات كاملة:** الانطلاقة (سرعة) • من يسبق؟ (جرس + كشف كلمة كلمة) • السلسلة الذهبية («ثبّت النقاط») • المواجهة (مبارزات إقصاء + موت مفاجئ، بثلاثة لاعبين فأكثر) • النهائي (اختيار فئات قوة وفخ، سباق إلى 100، خطف بـ 15).
- **تدريب فردي:** سباق الـ 60 ثانية مع أفضل نتيجة محفوظة.
- **تحدّي أونلاين:** غرف برمز من 6 محارف، 2–6 لاعبين على أجهزة منفصلة، 10 أسئلة، نقاط مباشرة ومكافأة لأول إجابة صحيحة.
- **بنك 1000 سؤال بثلاث لغات** عبر 22 فئة و4 مستويات صعوبة (كل فئة لها أسئلة صعبة/خبير)، مع إجابات وخيارات وتفسيرات وتلميحات مترجمة — بلا أي تكرار (يحرسه اختبار آلي).
- **تطبيع إجابات عربي** (تشكيل، همزات، تاء مربوطة، ألف مقصورة، «ال»، تسامح إملائي).
- **شاشة بطل** مع 5 إحصاءات (منها المستوى المتقن) وبطاقة مشاركة PNG وقصاصات — لقب النهائي يحسم البطل.
- **إحصاءات الجهاز:** ألعاب مكتملة، دقة، سجل أبطال.
- **إعدادات لعب:** سرعة المؤقت، نمط الإجابة (كتابة/خيارات)، نبرة المقدّم (فصحى/لبنانية)، حفظ آخر إعدادات.
- **وصولية:** مفاتيح داخل التطبيق (خط كبير، تباين عالٍ، تقليل حركة) + `prefers-reduced-motion` + ARIA.
- أصوات أصلية مولّدة بـ WebAudio، وضع كتم، ودعم تثبيت PWA.

## البنية

```
src/
  types/        النماذج (Question, PlayerState, Phase, ScoreEvent, QuestionPack, HostAction, ...)
  data/         بنك الأسئلة (1000 سؤال، 22 فئة) — questions.ts + extra*.ts + اختبار تفرّد
  lib/          answerNormalizer / scoreEngine / questionPicker / soundEngine / prefs / packIO
  engine/       gameEngine.ts — آلة حالات نقية (reducer) للجولات الخمس + اختباراتها
  store/        Zustand
  components/   TimerRing, Scoreboard, QuestionCard, AnswerPanel, Confetti, ConfirmModal
  screens/      Home, Setup, Game, Solo, Stats, Champion + rounds/ (1–5)
docs/
  SPEC.md       المواصفات الكاملة (الأونلاين، الإدارة، خارطة الطريق)
  DATABASE.sql  مخطط PostgreSQL/Supabase للنسخة الأونلاين
```

**التقنيات:** React 19 + TypeScript + Vite + Tailwind CSS v4 + Framer Motion + Zustand.

## التالي (انظر docs/SPEC.md)

تشغيل الجولات الخمس كاملة عبر الأجهزة مع جرس لحظي بختم خادمي ← لوحة المقدّم ← لوحة إدارة محمية للأسئلة والحزم ← أسئلة صوت/صورة بالتخزين السحابي ← طابور مراجعة ← الحزم المدفوعة والبطولات.

## إعداد التحدّي الأونلاين

1. أنشئ مشروع Supabase أو اختر المشروع الصحيح.
2. فعّل **Anonymous Sign-Ins** من Authentication → Providers.
3. طبّق migration الموجود في `supabase/migrations`.
4. انسخ `.env.example` إلى `.env.local` وضع رابط المشروع و**Publishable key** فقط.
5. شغّل `npm run dev` وافتح التطبيق على جهازين لاختبار الإنشاء والانضمام.

لا تضع `service_role` أو Secret key داخل متغيرات `VITE_*`.
